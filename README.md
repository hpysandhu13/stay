# stay

Stay is a React + Firebase app where users can:

- Sign up and sign in
- Manage personal profile pages
- Create hosting posts in a community feed
- Send direct messages to other users

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Copy `.env.example` to `.env` and add your Firebase project values.

3. Start the app:

   ```bash
   npm run dev
   ```

## Quality checks

```bash
npm run lint
npm run build
```
